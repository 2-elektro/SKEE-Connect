# SKEE Connect

SKEE Connect is a smart energy management platform developed by SKEESustain.

## Products

### SKEE Meter

Reads smart meter (P1) data and sends telemetry to the cloud.

### SKEE Hub

Controls and monitors heat pumps and thermal storage systems.

### SKEE Solar

Controls solar inverters and prevents unnecessary grid export.

## Features

* Smart meter integration
* Heat pump optimization
* Solar power optimization
* Cloud connectivity
* Remote support
* OTA firmware updates
* Mobile app integration
